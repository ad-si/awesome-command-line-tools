- [Blessed Contrib] - Build terminal dashboards

- [Terminal Table] - Ruby ASCII Table Generator, simple and feature rich.
- [CLI Chart] - Ansi Color Bar Charts.

[Blessed Contrib]: https://github.com/yaronn/blessed-contrib
